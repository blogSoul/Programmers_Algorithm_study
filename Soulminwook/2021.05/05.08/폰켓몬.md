# 📕 Solution

```java
import java.util.HashSet;

class Solution {
    public int solution(int[] nums) {
        HashSet<Integer> set = new HashSet<>();
        for(int num : nums){
            set.add(num);
        }
        return (set.size() < nums.length / 2) ? set.size() : nums.length / 2;
    }
}
```
